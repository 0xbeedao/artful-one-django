About me

Atom feeds

The main feed for my site combines my blog entries, my blogmarks and my collected quotations:

https://artful.one/atom/everything/

If you just want my longer form blog entries you can subscribe to this feed instead:

https://artful.one/atom/entries/

Or this feed for just my links (excluding quotes):

https://artful.one/atom/links/

Every tag on my blog has its own feed. You can subscribe to those by adding .atom to the URL to the tag page.

For example, to subscribe to just my content about ai, use the following:

https://artful.one/tags/ai.atom

About this site

This site is a custom web application built using Django. It is based on Simon Willison's Blog, but with entirely new templates, CSS and the image carousel functionality replaced with a custom implementation that I'm quite pleased with. The source code can be found on GitHub at 0xbeedao/artful-one-django.
